YUKI

Guest Concierge Agent

✅ Must-Have Capabilities

Answer Hotel Questions Available
FAQ about amenities, hours, policies from knowledge base
View Booking Details Available
Look up reservations, check-in times, room assignments

Send pre-arrival info, welcome messages, checkout reminders automatically

💡 Requires: Webhook/event system + messaging API integration
Omnichannel Communication Blocked
WhatsApp, LINE, SMS, Email unified inbox with AI responses
⚠ 73% of guests prefer messaging over phone calls
💡 Requires: Twilio/MessageBird integration + conversation API
Guest Sentiment Analysis Blocked
Detect unhappy guests from message tone, escalate proactively
⚠ Negative reviews preventable with early intervention
💡 Requires: NLP sentiment model + guest interaction history
Personalized Recommendations Partial
AI-driven suggestions based on guest history, preferences, weather
⚠ Generic recommendations miss upsell opportunities
💡 Requires: Guest profile API + preference tracking

🚫 Blocked by RoomBoss

Real-time Service Request Tracking Blocked

⚠ No task management API for request lifecycle
In-App Chat History Blocked
Persistent conversation threads across guest stays
⚠ No conversation storage or guest messaging API

Interactive in-room tablet/app with all hotel info
⚠ No content management API for dynamic updates
Review & Feedback Collection Blocked
Post-stay surveys, NPS tracking, review prompts
⚠ No feedback/survey API or guest satisfaction tracking
Lost & Found Management Blocked
Report lost items, track found items, notify guests
