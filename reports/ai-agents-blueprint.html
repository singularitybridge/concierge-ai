AI Agent Roster

🙋
YUKI
Guest Concierge Agent

The primary guest-facing AI assistant. YUKI embodies omotenashi (Japanese hospitality), providing personalized service 24/7. Handles guest inquiries, service requests, recommendations, and ensures every guest feels welcomed and cared for throughout their stay.

Data Sources

🔌 Hotel API (Read)
🔌 GS Purchasing API
📚 Guest Services Guide
📚 Hotel Info & Policies

Supported Actions

💬
Answer Guest Questions
Hotel info, amenities, hours, dining, onsen etiquette, local area
🍽
Make Recommendations
Dining suggestions, activities, spa treatments, local excursions
🔔
Process Service Requests
Room service, housekeeping, amenities, laundry, wake-up calls
📝
View Booking Details
Check-in/out times, room info, invoice status, special requests
🌟
Handle Special Occasions
Birthday/anniversary arrangements, VIP recognition, surprise setups

Example Requests

"What time does the onsen close?" "Can I get extra towels?" "Book dinner at Yuki restaurant" "What's the WiFi password?" "Recommend a sake for tonight"
🌸
SAKURA
Reservations Agent

The booking lifecycle specialist. SAKURA manages the complete reservation journey from initial inquiry to final checkout. Handles availability searches, rate quotes, booking modifications, cancellations, and payment processing with full policy awareness.

Data Sources

🔌 Hotel API (Full CRUD)
🔌 OTA Rates API
📚 Operations Manual
📚 Room Configurations

Supported Actions

🔍
Check Availability
Search rooms by dates, guests, room type with real-time inventory
💳
Get Rate Quotes
Pricing by rate plan, seasonal rates, OTA vs direct pricing
✅
Create Bookings
New reservations with guest details, rate plans, special requests
✏
Modify Reservations
Date changes, room upgrades, add guests, update guest info
❌
Process Cancellations
Cancel with policy enforcement (14-day full, 7-day 50%, <7 no refund)
💳
Register Payments
Deposits, balance payments, refund processing

Example Requests

"Check availability for 2 adults, March 15-20" "Book the Onsen Suite for the Tanaka family" "What's the cancellation policy?" "Change my checkout to the 18th" "Process a ¥60,000 deposit"

🔌 Hotel API (listBookings)
📚 Operations Manual
📚 Staff Protocols

🚿
Update Room Status

Manage Arrivals/Departures

📝
Coordinate Housekeeping
Task assignments, completion tracking, priority queues
🛠
Log Maintenance Issues
Report issues, track resolution, escalate urgent items

Pre-Registration Processing
Mark guests as pre-registered, verify documents, prepare rooms

Photos for ski passes, lift tickets, ID requirements

Experience Packages
Create bundled activity packages, process payments

💵
Rate Management

👥
Guest Analytics
VIP identification, repeat guest tracking, nationality mix, stay patterns
💡
Strategic Recommendations
Pricing suggestions, demand forecasting, promotional opportunities

Example Requests

"What's our occupancy next week?" "Show revenue for December" "Update rates for Golden Week" "Who are our VIP arrivals?" "Should we run a promotion?"

Agent Routing Logic

How requests are intelligently routed to the appropriate agent:

Request Category Primary Agent Handoff Triggers
General questions, service requests, recommendations YUKI Booking changes → SAKURA, Activities → HANA
Availability, pricing, booking CRUD, payments SAKURA Activity add-ons → HANA, Post-booking service → YUKI
Room status, housekeeping, arrivals, staff tasks KENJI Guest issues → YUKI, Booking lookup → SAKURA
Ski rentals, lessons, tours, transportation HANA Payment issues → SAKURA, General questions → YUKI
Reports, analytics, rate management, forecasting REN Rate changes → confirm with Management

Implementation Recommendations

✅ Phase 1: Quick Wins

  • Deploy YUKI for guest-facing chat
  • Deploy SAKURA for reservation management
  • Integrate voice via Vapi/ElevenLabs

🛠 Phase 2: Operations

  • Deploy KENJI for staff portal
  • Build housekeeping workflow automation
  • Integrate with existing task system

🎿 Phase 3: Experiences

  • Deploy HANA for activities booking
  • Connect GS Purchasing API fully
  • Build experience package builder

📊 Phase 4: Intelligence

  • Deploy REN with data warehouse
  • Build ETL for historical analytics
  • Enable rate push automation

⚠️ Known Limitations (API Gaps)

The RoomBoss API does not provide: native reporting/analytics endpoints, historical data queries, rate plan creation, room type configuration, channel manager setup, or user management. REN agent requires a custom data warehouse layer for full BI capabilities.
