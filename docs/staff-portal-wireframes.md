## Responsive Breakpoints

### Small Phone (iPhone SE - 375px)
```
Adjustments:
- Stats: 2 columns (no change)
- Task cards: Full width - 16px padding
- Font sizes: Minimum (no scaling down)
- Touch targets: 56px (absolute minimum)
```

### Large Phone (iPhone Pro Max - 428px)
```
Enhancements:
- Stats: 2 columns with more padding
- Task cards: Slightly larger padding
- Font sizes: Can scale up 10%
- Touch targets: 64px (more comfortable)
```

### Tablet (768px+)
```
Layout Changes:
- 2-column task grid
- Sidebar with filters always visible
- Stats: 4 columns
- Task detail: Modal instead of full screen
```
